## Ideia 1: Institucional Autoritário — "Rigor Acadêmico"

**Design Movement:** Modernismo Institucional com influência do Swiss Style

**Core Principles:**
- Hierarquia tipográfica rígida e clara, transmitindo seriedade e confiança
- Uso de linhas verticais e divisores como elementos estruturantes
- Contraste forte entre vermelho institucional e branco limpo
- Dados e informações sempre em destaque visual

**Color Philosophy:**
- Vermelho #8b1120 como cor primária de autoridade e identidade
- Branco puro como espaço de respiro e leitura
- Azul #1f3a5f como cor de confiança e dados
- Cinza muito claro (#f7f7f7) para fundos de seções alternadas

**Layout Paradigm:**
- Assimétrico com coluna lateral vermelha fixa na hero
- Grid de 12 colunas com módulos de conteúdo bem definidos
- Seções com bordas esquerdas coloridas como marcadores visuais

**Signature Elements:**
- Barra lateral vermelha vertical na hero section
- Tipografia condensada em caixa alta para títulos de seção
- Ícones de linha fina em azul escuro

**Interaction Philosophy:**
- Hover com sublinhado deslizante em vermelho
- Transições suaves de 200ms para estados de foco
- Cards com elevação sutil ao hover

**Animation:**
- Entrada de elementos com slide-in da esquerda
- Contadores animados para estatísticas
- Fade suave entre seções

**Typography System:**
- Títulos: Barlow Condensed Bold (caixa alta)
- Corpo: Source Serif 4 (legibilidade acadêmica)
- UI/Labels: DM Sans Medium

## Ideia 2: Editorial Moderno — "Clareza Pedagógica" ✅ ESCOLHIDA

**Design Movement:** Editorial Contemporâneo com influência de design escandinavo aplicado ao setor educacional

**Core Principles:**
- Tipografia expressiva como elemento visual principal
- Espaço branco generoso para respiração e foco
- Vermelho como acento poderoso, não como fundo dominante
- Estrutura assimétrica com âncoras visuais fortes

**Color Philosophy:**
- Vermelho #8b1120 como cor de identidade e chamada à ação — usado com parcimônia e impacto
- Branco #ffffff como base dominante, transmitindo transparência e clareza
- Azul #1f3a5f como cor de dados, confiança e elementos secundários
- Off-white #f9f6f2 para fundos de seções alternadas, criando profundidade sutil

**Layout Paradigm:**
- Hero com divisão diagonal entre vermelho e branco
- Layout editorial com texto grande e imagens em proporções áureas
- Cards com borda esquerda colorida como identificadores de categoria
- Seções alternando entre fundo branco e off-white

**Signature Elements:**
- Divisor diagonal vermelho/branco na hero
- Números grandes em vermelho como elementos decorativos de estatísticas
- Linha horizontal vermelha fina como separador de seções

**Interaction Philosophy:**
- Botões com preenchimento sólido vermelho e hover em azul escuro
- Cards com sombra crescente ao hover
- Links com sublinhado animado em vermelho

**Animation:**
- Entrada de seções com fade + translateY(20px)
- Números de estatísticas com contagem animada
- Smooth scroll entre seções

**Typography System:**
- Títulos: Playfair Display Bold (elegância editorial)
- Subtítulos: Raleway SemiBold
- Corpo: Inter Regular (legibilidade digital)
- Dados/Números: Barlow Condensed Bold

## Ideia 3: Tecnológico Institucional — "Plataforma Digital"

**Design Movement:** Tech-Institutional com influência de design de SaaS educacional

**Core Principles:**
- Interface que comunica tecnologia e confiança simultaneamente
- Gradientes sutis em vermelho para profundidade
- Elementos de UI que remetem a dashboards e dados
- Estrutura de produto digital com navegação clara

**Color Philosophy:**
- Vermelho #8b1120 em gradiente para #c41e3a como cor de marca
- Azul #1f3a5f como cor de dados e elementos técnicos
- Branco com sombras suaves para cards e painéis
- Fundo levemente acinzentado (#f4f5f7) para profundidade

**Layout Paradigm:**
- Hero com mockup de dashboard flutuante
- Grid de features em 3 colunas com ícones grandes
- Seção de benefícios com layout alternado imagem/texto

**Signature Elements:**
- Mockup de interface/dashboard na hero
- Ícones em containers com fundo colorido suave
- Badges e tags para categorização de funcionalidades

**Interaction Philosophy:**
- Animações de entrada escalonadas para listas de features
- Hover com mudança de cor de ícone
- Scroll-triggered animations para estatísticas

**Animation:**
- Stagger animation para cards de features
- Parallax sutil no hero
- Progress bars animadas para estatísticas

**Typography System:**
- Títulos: Plus Jakarta Sans Bold
- Corpo: Nunito Regular
- Código/Dados: JetBrains Mono
